Ubuntu

Ubuntu應該算是最流行的Linux發行版，雖然Ubuntu Unity的熱門程度似乎不如Mint和elementary OS，但後面兩個也只不過是基於Ubuntu的修改版，共同的問題很容易在社群上找的解決方案。
第一次用Ubuntu應該是在國中，那時候家裡有台有點年紀的筆電，想說裝Ubuntu會不會讓他快一點，結果Unity太不直覺反而沒人想用囧。
這次在跳的Ubuntu的時候，感覺最大的不同是瀏覽器跟雲端應用程式的進步。以前大部份應用是以獨立GUI為主，當時GUI的開發光有OSX版就很了不起了。而Linux上發行版百家爭鳴，結果就是沒人想開發，想用只能仰賴Wine和VM。而且Wine那時也還相當不成熟。現在像Line、Office即使沒有出Linux版，在Chrome上也能正常使用，甚至像Curse語音的Web版介面，可以說比Windows的GUI還要好用。Google對Linux真的很有愛，以前在Windows從來沒感覺Chrome應用程式可以做這麼多事XD。

Gnome

不得不說，Unity真的是一個違反人類的設計，又醜又難用。
之前有試過Mint Cinnamon，UI基本上是照著Gnome2和Windows的方向在設計，使用起來算方便上手，但像是字型一類的設定不容易修改。
後來在VM上裝了一次Ubuntu Gnome，第一次發現原來Gnome 3這麼漂亮（不過Gnome 3特效太多，對硬體太差的電腦真的算是負擔）。後來就轉投Gnome 3了，不過Gnome 3跟中文的相容性也真的頗慘，大部份時間幾乎都在解決不相容的問題，這也是為什麼會想刻意整理一篇出來備忘。

設定交大為更新伺服器

軟體和更新→下載自：ubuntu.cs.nctu.edu.tw
雖然預設是台灣，不過清大連交大比較快，畢竟在隔壁。

更新套件

將全域設為中文環境

設定→地區和語言

這時候系統會要你裝缺少的套件，先安裝下去。

語言→漢語(台灣)→完成

系統會要求重新啟動，選擇立即重新啟動，就完成了。

設定輸入法為hime或gcin

http://goodjack.blogspot.tw/2013/08/linux-phonetic-setting.html

fcitx跟ibus是系統預設中文輸入法，雖然在gnome下有大量bug存在，但會在前面設定中文環境時被自動安裝，如果不移除，就算安裝gcin或hime也不能用。
另外，hime在Gnome下似乎有不能輸出單一文字的問題，而gcin相容性則比hime要來得差，自己要評估一下取捨。

原文中有安裝最新版本hime的方法，有興趣可以點進去看
